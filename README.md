# Obsidian自动化脚本库

Obsidian手动操作太繁琐？
**脚本库帮你自动化、云备份。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()
[![Obsidian](https://img.shields.io/badge/Obsidian-自动化-purple)]()

## 操作不再"手动"

- **健康检测**：8项知识库健康指标
- **云备份配置**：阿里云OSS/腾讯云COS/AWS S3
- **脚本管理**：自动化脚本安装指南
- **状态监控**：知识库运行状态大盘

## 怎么用

打开 `install.html` → 选择脚本 → 查看安装指南
